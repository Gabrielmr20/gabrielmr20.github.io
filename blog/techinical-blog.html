Control flow and loops

Control flow is the order in which a process is executed. In programming the code is executed from the first line to the last line. Unless changes are made to change the execution order.



Loops like the name says are instructions that keeping repeating/looping until a condition is achieved. For example, keeps fulfilling fridge until the fridge is full.



What is the DOM?

The DOM (Document Object Model) is a programming interface. It takes HTML content and convert that into a object so JavaScript can access and modify the code.


Arrays vs Objects - Accessing data

Arrays and objects are frequently used in JavaScript, understanding how they work and how to access content inside them is vital.



An array is an ordered collection of elements mainly values or variables that are identified by an index (index indicates the position of the element in the array)



Creating an array in JavaScript is simple, you can check how it is done here:



let array = ["item1", "item2", "item3"]
array[1]

// This will return "item2" because the
// index of an array always starts from 0

The difference of an object and an array is that the data inside the object are known as properties that consist of a keys and values. The properties can be manipulated using .dot notation and [].


let person = {
name: "Gabriel"
age: 100
}

person.name 
// This will return Gabriel using dot notation

person["name"] 
// This will return Gabriel using the [] notation
What are functions?

Functions in JavaScript are a set of statements that performs a task. It allows to call that piece of code instead writing that over and over.
