The difference between Margin, Border, and Padding

Margin is how much space is around an object.

Padding is how much space is within an object side and its contents.

Border is between the margin and padding of an object. Only counted when given an actual width otherwise, it’s zero.

The place where it all matters is defining width. Say you have two yellow shoe boxes that are 30cm wide and can get as long as you need. No lid and you’re looking down into one of the boxes.

The border is the box edge and is 2cm thick and red.

Imagine the paper stuffing around the shoe is the padding and is 4cm thick and transparent (padding don’t have color associated to it).

The margin 4cm. This is the space between the boxes and also don’t have a color associated to it.

So here is some code exemple of how it works, on this code box is a .CSS class that willdefine the "box size":

.shoe-box {
  width: 300px;
  height: auto;
  background-color: yellow;
  border: 20px solid red;
  padding: 40px;
  margin: 40px;
  word-wrap: break-word;
}

Here is the HTML code:

<div class="shoe-box">
  <p>xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx</p>
</div>
